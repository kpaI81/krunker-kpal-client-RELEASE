## Features
- Crosshair overlay
- Custom Image Crosshair
- Custom Scope
- Customised Matchmaking
- Unlimited FPS
- Window Capture (DX9 Mode)
- Force Color Profile

## Keybinds
- F4 - Finds random new match
- F5 - Refreshes page
- F11 - Fullscreen
- Alt+F4 - Closes window
- Tab - Toggles mini menu
